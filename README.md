# Birthday Dash Deluxe

一個適合生日派對的大螢幕多人競速遊戲，支援手機掃 QR Code 加入，並用原創像素風做出復古橫向卷軸的派對感。

## 新版特色

- 原創像素風橫向卷軸舞台，視覺更接近經典電玩大螢幕比賽感。
- 房主按下開始後會先進入 `3 2 1 GO!` 倒數。
- 內建簡單 8-bit 風格倒數音效、按鍵音和勝利音效。
- 增加更多預設頭像，可讓玩家選擇不同像素角色。
- 舞台有星星磚塊加速區、泥巴陷阱區、像素水管和終點城堡。
- 玩家手機端會同步顯示自己的賽道預覽。

## 玩法

- 主持人在電腦、平板或投影設備開啟首頁。
- 畫面會自動建立房間與 QR Code。
- 玩家用手機掃描 QR Code，輸入名字、選角色後加入。
- 房主按下開始比賽後，全場先倒數再開跑。
- 玩家在手機上狂按 `RUN` 按鈕，把角色一路衝向終點城堡。
- 第一個到終點的玩家獲勝。

## 部署到 GitHub Pages

1. 把 `index.html`、`styles.css`、`app.js`、`README.md` 放到 repository 根目錄。
2. 在 GitHub repository 的 `Settings` -> `Pages`。
3. 將 `Source` 設為 `Deploy from a branch`。
4. 分支選 `main`，資料夾選 `/ (root)`。
5. 儲存後等待 GitHub Pages 發佈。

## 注意事項

- 這個版本是純前端靜態網站，適合直接部署到 GitHub Pages。
- 即時多人連線使用 PeerJS 公用 broker 進行瀏覽器協調，所以不需要自架後端。
- 音效使用瀏覽器 Web Audio API 產生，通常要在使用者互動後才會開始播放。
- 若現場網路不穩，手機與主機的連線流暢度會受影響。
